VDI

Virtual Desktop Infrastructure (VDI) is ability to run user desktop environment as a VM with remote access. VDI is major accelerator and cost saver approach in scope of End User Computing (EUC) domain. Leveraging VDI solution companies of all sizes can benefit from great security posture reducing overall operational cost and eliminating idle usage of expensive hardware such as graphical accelerators. VDI solution might be deployed completely in-house or consumed as combination of managed services from public cloud vendor on pay-as-you-go model.
